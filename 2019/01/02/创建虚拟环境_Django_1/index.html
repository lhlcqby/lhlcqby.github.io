MVT/MTV (Django模型）

M ：  modelsp.py     模型层
V:    views.py     写业务逻辑的地方
T  : templates . 模板，定义页面HTML

MVT 模式 是由MVC 模式演变出来的

MVC 模式是所有框架遵循的模式

M：模型层
V: 模板  HTML 页面
C  controller . 控制器，写业务逻辑

今天主要内容 – 虚拟环境的创建

1.安装虚拟环境(CMD)

pip install virtualenv

2 指定位置 创建虚拟环境
（cd D:/env)
virtualenv –no-site-pages -p E:\python\python.exe(python 编译器位置）django8(虚拟名）

3 启动虚拟环境

activate
如果要停止 命令：deactivate

4 返回到Workspace(设定好的专门装代码的文件夹）创建项目

（django8)E:\workspace : django-admin startproject day01(项目名）

注意1 ：一个虚拟环境只能装一个版本的Django。
注意2 ： 以后完成项目后 端口设置成全部可访问的
0.0.0.0 :80    (端口默认都是80）

疑惑2

###：修改vim配置 /etc/vim/vimrc
set nocompatible 不使用vi默认键盘布局，如果不配置，在编辑模式下按方向键会输入字符。
set mouse = a （支持鼠标）

set showmatch 括号匹配模式
set ruler 显示状态行
set tabstop=4 tab键为4个空格
set shiftwidth=4
set softtabstop=4
syntax on　　　　　 　语法高亮

set nu 　显示行号
set cindent C语言格式对齐
set autoindent 自动对齐

set smartindent 智能对齐

固态硬盘和机械硬盘

SSD（固态硬盘）和HDD（机械硬盘）之间的巨大差距来源于它们的结构，SSD为全电子结构，采用集成电路存储技术，使用存储芯片加上寻址控制器来组成的硬盘，类似于U盘技术，没有任何机械运动部件。传统的机械硬盘HDD采用高速旋转的磁盘来存储数据，通过磁头来读写，这一机械运动过程中带有延迟、并且无法同时迸发多向读写数据。

丹尼斯

丹尼斯·里奇，C语言之父，UNIX之父。曾担任朗讯科技公司贝尔实验室下属的计算机科学研究中心系统软件研究部的主任一职。1978年与布莱恩·科尔尼干（Brian W. Kernighan）一起出版了名著《C程序设计语言（The C Programming Language）》。此书已翻译成多种语言，被誉为c语言的圣经。2011年10月12日（北京时间为10月13日），丹尼斯·里奇去世，享年70岁。
