1 布隆过滤器
布隆过滤器（Bloom Filter）是1970年由布隆提出的。它实际上是一个很长的二进制向量和一系列随机映射函数。布隆过滤器可以用于检索一个元素是否在一个集合中。它的优点是空间效率和查询时间都比一般的算法要好的多，缺点是有一定的误识别率和删除困难。
如果想判断一个元素是不是在一个集合里，一般想到的是将所有元素保存起来，然后通过比较确定。链表，树等等数据结构都是这种思路. 但是随着集合中元素的增加，我们需要的存储空间越来越大，检索速度也越来越慢。不过世界上还有一种叫作散列表（又叫哈希表，Hash table）的数据结构。它可以通过一个Hash函数将一个元素映射成一个位阵列（Bit Array）中的一个点。这样一来，我们只要看看这个点是不是 1 就知道可以集合中有没有它了。这就是布隆过滤器的基本思想

#2 开源镜像
“镜像网站”（英文:Mirror sites [1] ），又译作““镜像站点” [1]
搜狐开源镜像站：http://mirrors.sohu.com/ 

既然是相同的网站内容，为什么要放在不同的网站服务器里，这难道就是“有钱任性”吗。其实不然，看看下面列举的“网站镜像”的几个好处吧。

每个网站服务器的稳定性不是100%的，还有如果网站受到一些DDOS攻击，就会导致网站打不开，有了镜像网站，就不用怕了，还有其它网站同样可以浏览，这个功能有点像网站备份。
如果你做的网站足够大，每天有很大的流量，特别一些下载类型网站，每天有大量的用户去下载网站上的资源，如果这些资源下载地址只使用一个网站的话，对于网站服务器的负担是很重的。这时我们可以引导用户到“网站镜像”上下载，来分担主网站的负担。这就是“镜像下载”

时间装饰器

def timeit(fun)
    def wrapper(x,y):
        str_time = time.time()
        fun(x,y)
        end_time = time.time()
        print("%s 函数运行时间为$s" %(fun.__name__,end_time- str_time))
    return wrapper

@timeit
def add(x,y):
    return x+y

print(add(1,3))

import time
def echoRuntime(func):
    def wrapper(*args, **kwargs):
        startTime = time.time()
        func(*args, **kwargs)
        endTime = time.time()
        msecs = (endTime - startTime)
        print(func.__name__+" running time is %.2f s" %msecs)
    return wrapper
